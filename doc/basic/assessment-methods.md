# Assessment Methods Specification

## Purpose
Define standard assessment methods for key decision points in the lifecycle.

## 1. Acquisition Case
- **Goal:** Determine whether to proceed with acquiring a new asset/opportunity.
- **Checks:**
  - Strategic fit
  - Cost/benefit baseline
  - Risk and compliance screening
- **Decision outputs:** Approve, Request revision, Reject.

## 2. Strategic Decision Case
- **Goal:** Evaluate major direction changes or investments.
- **Checks:**
  - Alignment with long-term objectives
  - Scenario comparison and trade-offs
  - Resource and capability readiness
- **Decision outputs:** Commit, Stage-gate, Defer.

## 3. Exit Case
- **Goal:** Decide whether to divest, sunset, or terminate.
- **Checks:**
  - Performance vs. threshold
  - Exit impact and dependency mapping
  - Transition and communication plan quality
- **Decision outputs:** Exit now, Managed transition, Continue with corrective action.
